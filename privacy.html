Privacy Policy / 隐私政策

Your data stays on your device.

Last updated: March 18, 2026 / 最后更新：2026 年 3 月 18 日

FocuX has zero servers, zero cloud storage, zero analytics, and zero tracking. All EMG data, AI analysis, and session records are processed and stored exclusively on your iPhone.

1. Data Collection / 信息收集

FocuX collects the following data, all stored locally on your device:

  • EMG signals — Surface electromyography data captured via BLE from WAVELETECH wearable band. Processed on-device in real-time.
  • Session records — Duration, timestamps, Stamina scores, and three-axis metrics. Stored in local SwiftData database.
  • Device identifiers — Basic BLE device identifiers used solely for sensor pairing.
  • AI coaching summaries — Generated entirely on-device using Apple Foundation Models. No data is sent to any server or cloud API.

2. On-Device AI / 端侧 AI

FocuX uses Apple Foundation Models to provide personalized AI coaching summaries. This processing happens entirely on your iPhone:

  • All AI inference runs locally via Apple Foundation Models framework.
  • No EMG data, session data, or personal information is sent to Apple, OpenAI, or any third-party AI service.
  • When Foundation Models is unavailable on your device, FocuX falls back to a built-in rule engine — still fully offline.

3. Data Usage / 数据使用

Your data is used exclusively for:

  • Real-time Stamina Score computation and display
  • Session summaries and historical analysis
  • On-device AI coaching and personalized recommendations
  • Live Activity lock screen status display
  • Home screen widget data display

4. App Group Data Sharing / App Group 数据共享

FocuX uses an App Group (group.com.fluxchi.app) to share summary data between the main app and home screen widgets. This sharing occurs entirely on-device — no network transmission involved.

FocuX 使用 App Group 在主 App 和桌面 Widget 之间共享摘要数据（如训练次数、续航均值），此过程完全在设备本地完成。

5. Data Storage & Security / 存储与安全

All data resides on your device. FocuX does not transmit EMG data, session records, AI summaries, or any personal information to any server or cloud service.

WiFi mode data transmission occurs only within your local network (LAN).

6. Third-Party Services / 第三方服务

FocuX integrates zero third-party analytics, advertising, or tracking services. No Firebase, no Google Analytics, no cloud AI APIs, no similar tools.

7. Data Deletion / 数据删除

Swipe left on any session in the History view to delete it. Uninstalling the app removes all local data permanently.

8. Children's Privacy / 儿童隐私

FocuX is not directed at children under 13. We do not knowingly collect personal information from children.

9. Permissions / 权限说明

  • Bluetooth — Required. Connects to WAVELETECH EMG wearable band for signal reception.
  • Local Network — Optional. Enables WiFi connection to desktop EMG data server for development and testing.

10. Changes / 政策变更

Any significant changes to this policy will be communicated through the app or this page.
